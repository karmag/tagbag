    Alt + V              -> mark/grid-3-2
    Alt + T              -> mark/grid-4-0

    Alt + B              -> mark/grid-4-2
    Control + Enter      -> mode/command
    Tab                  -> mode/single

    Enter                -> press-enter
    Control + R          -> refresh
    End                  -> scroll/bottom
    Next                 -> scroll/page-down
